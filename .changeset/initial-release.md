Initial v1.0.0 release of the n8n community node for Harvest API integration. This release includes:

- Complete Harvest API v2 integration with OpenAPI-generated types
- Support for all major Harvest resources: Projects, Tasks, Time Entries, Clients, Invoices, Expenses, Users, and Reports
- Full CRUD operations for all supported resources
- Type-safe TypeScript implementation
- Comprehensive error handling and validation
- Ready for production use in n8n workflows
